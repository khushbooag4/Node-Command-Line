# Node-Command-Line
To intiate the process type 'npm init' command in the terminal
Now play with the terminl the way you want

ADD COMMAND
- cli3 add
- cli3 a

FIND COMMAND | name can be firstname or lastname 
- cli3 find name 
- cli3 f name
  
UPDATE COMMAND
- cli3 update <_id>
- cli3 u <_id>
  
REMOVE COMMAND
- cli3 remove <_id>
- cli3 r <_id>
  
LIST COMMAND
- cli3 list
- cli3 l
  
KNOW THE VERSION
- cli3 --version
